Папки: 
css - папка со стилями

Файлы:

admin.php - скрипт регистрации, также добавляет куки юзера.
admin_lk.php - выводит . 
exit.php - обнуляет куки, чтобы выйти из админки.
form_admin.php - форма авторизации админа. Админ добавляется только через БД.
holiday.php - скрипт определения праздника по дате.
holiday_day.php - скрипт определения даты праздника по названию.
index.php - главная страница, определяет, если праздник на текущую дату.
new_holi.php - скрипт, позволяет добавлять новые праздники в бд через админку. 
404.php - страница ошибки 404, выводится только если в админку пытается войти не админ.
zd12-2.sql - бд, импортируй в phpmyadmin.

Все взаимодействия с бд производятся через команды а-ля  $mysql = new mysqli('localhost','root','','kurs');
Если у тебя другие названия пользователя или есть пароль - конструкцию надо будет соответствующе изменить, например:
	$mysql = new mysqli('localhost','ПОЛЬЗОВАТЕЛЬ','ПАРОЛЬ','НАЗВАНИЕ БД');

Данные по праздникам берутся полностью из бд и редактируются. Из минусов - поиск праздника должен быть предельно конкретным, вплоть до регистра и точек в дате.
